● 커밋 이름 작성 요령

=> eternal_return-v + (날짜) + a~z
     변동사항 : 추가했거나 바뀐 내용을 적어주세요
        
예시)     eternal_return-v240529a
              - 새로운 스코어보드 추가
              - 하이퍼루프 상점기능 버그 수정
              - ....

● 주의할 점

1. 브렌치를 바꾸기 전에 커밋을 하거나 변경 내용을 적어두지 말아주세요!
2. 새로운 버전이 나오면 패치를 눌러서 확인해주세요!
3. 공유 폴더와 개인 폴더를 헷갈리지 마세요!

● 개발규칙
 -  map 파일을 업로드 할때에는 데이터팩 파일 안을 비워둔다
 - 리로스팩을 업로드 할 때(feature)를 업로드 하기 전에 무조건 fetch를 눌러준다
 - merge를 하기 전에 겹치는 코드가 존재하면 다른 개발자와 미리 상의한다
