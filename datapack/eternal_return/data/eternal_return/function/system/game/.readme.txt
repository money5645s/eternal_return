
------- 사용되는 스코어보드와 스토리지 설명 -------

> 스코어보드

- 타이머
    => 게임을 시작할 때 상단에 표기되는 타이머 변수
    => 각각 틱, 초, 분, 낮밤, 일차
    #timer.tick CT1
    #timer.sec CT1
    #timer.min CT1
    #timer.halfday CT1
    #timer.day CT1


- 플레이어 및 팀 수
    => 솔로에서는 team과 player는 같은 값을 공유
    #game.player CT1
    #gmae.team CT1

- 부활
    #game.resurrection CT1
    #gmae.respawn CT1

- 금지구역 시간
    @s CT3



> 스토리지

- 상단 UI에 표시되는 구역
    => 우선 map_data 스토리지에 있는 플레이어가 위치한 구역의 정보를 개인 스토리지(player_data)에 받아온다
    => 개인 스토리지에 저장된 구역정보의 타입을 분류한다
    => 분류가 완료된 구역 정보를 색을 입혀 개인 스토리지 place_text에 저장한다
    => 최종적으로 상단 UI에서 place_text를 출력한다
